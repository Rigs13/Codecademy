HTML

Common HTML Tags

Tags Description
<head> Used to contain all the head elements in the HTML file, such as title, style, meta, etc...
<body> Defines the body of the HTML document (main content).
<h1> h1 through h6 defines heading tags based on importance (lowest to highest). Aids in creating the flow of the content.
<p> Defines a paragraph element within the HTML document. Usually placed following headings.
<a> Denotes anchors for wrapping around elements. Handy for adding clickable links to specific sections with href attribute.
<ul> Used with <li> tags for creating an unordered list of elements. Can also be ordered list with <ol> tags.
<!--Statement--> Common notation for adding comments into a HTML document.
<img> Used to add images directly into HTML documents.
<link> Used to link content from an external source. Helpful with CSS styling.

Common HTML Structures

Table HTML Format

Tags Description
<table> Starts the table element for creation. Must include closing tag after table is created.
<thead> Creates a header section for the table, common for creating labels for the table.
<tr> Creates a row for the <td> to store table data.
<td> This tag is used to store the data within the context of the table.
<th> Similar to a <td> tag, except for creating label headings.
<tfoot> Similar to the <thead>, used to create ending labels for the bottom of the table (Totals, etc...).
colspan="" Attribute that allows for a column to span more than one section.
